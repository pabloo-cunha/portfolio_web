Ferramente de Controle de Jornada

Este projeto foi feito em grupo, onde eu atuei principalmente como Scrumm Master do meu time.
O projeto consiste em uma ferramenta de controle de jornada de trabalho, onde o usuário pode registrar suas horas trabalhadas e gerar relatórios.
Para o administrador, é possível gerenciar os usuários e visualizar os relatórios de todos os usuários.
O projeto foi construído utilizado Java e Spring Boot para o backend e JS para front-end.

Gerenciamento das Oracle Partner Track

A Oracle Partner Tracker é uma plataforma moderna e inteligente de gerenciamento e análise de dados, capaz de interpretar, organizar e representar os dados do sistema OPN da empresa parceira Oracle. Entre os objetivos principais do projeto, se encontram a modernização do acompanhamento das empresas parceiras Oracle, assim como a visualização de dados de forma inteligente, para facilitar a identificação de melhorias e de conclusões estratégicas.
